# Task 03: Real-time Chat System

**Objective**: Build a robust communication layer using Supabase Realtime.

## 1. Core Chat Features
- **Scope**: Support both **1:1 Direct Messages** and **Group Chats** (Study Rooms).
- **Technology**: Supabase Realtime.

## 2. Rich Media & Events
- **File Support**: Enable Image uploading to Storage and display in chat.
- **System Automation**:
  - The system must automatically inject messages into the chat stream for events:
    - Member Join/Leave.
    - Member Check-in (verified attendance).

## 3. Deliverables
- Chat Interface (Mobile optimized).
- Realtime subscription hooks.
- Storage bucket configuration for chat images.
- System message generator utility.
